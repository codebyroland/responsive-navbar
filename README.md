### Installing Navbar

To install **Responsive Navbar**, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/rolandaayo/responsive-navbar.git
```

Windows:

```bash
git clone https://github.com/rolandaayo/responsive-navbar.git
```
